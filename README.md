# Projet Power BI – Analyse et Visualisation des Données

## Structure du projet

Le projet est organisé selon l'arborescence suivante :
```
.
├── data/
├── fichier_PowerBi/
├── rapport/
├── videos/
├── python_graphiques.ipynb
```

## Détails des dossiers et fichiers

### `data/`
Contient les fichiers Excel correspondant aux versions finales :
* des tables de dimensions
* À la table de faits

Ces fichiers sont le résultat de l'ETL établi dans Power BI et utilisé dans les tables de bords sur Power BI et le script .ipynb

### `fichier_PowerBi/`
Contient le fichier .pbi (Power BI Desktop) :
* Inclut le modèle de données
* Les transformations
* Les tableaux de bord et visualisations interactives

### `rapport/`
Contient le rapport final au format PDF :
* Description détaillée du projet
* Méthodologie
* Analyses réalisées
* Interprétation des résultats

### `videos/`
Contient une vidéo de présentation divisée en 4 parties (de partie 1 à 4) :
* Présentation du travail réalisé
* Explication du tableau de bord Power BI

### `python_graphiques.ipynb`
Notebook Jupyter contenant :
* Les scripts Python utilisés pour la génération de graphiques
